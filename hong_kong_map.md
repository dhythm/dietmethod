# Sightseeing
==================================================  
## Hong Kong Island
### Central
+ [x] Victoria peak
  - 香港の街が一望できる名所
  - ピークトラムは並ぶ
  - 複数人だとタクシーで行くのもあり（ぼったくりタクシーには注意）
+ [x] ランカイフォン
  - パリピの溜まり場
  - ウェーイウェーイ
+ [x] PMQ
  - アンティークなショッピングモール
  - 長時間いても飽きないのでオススメ
+ [x] 大館
  - 刑務所を改装したおしゃれ建物
+ [x] Pottinger Street
  - 石畳みの階段
  - 一瞬で終わるので、PMQや大館とセットでどうぞ
+ [x] 国際金融中心
  - 通称IFC
  - IFC mollでお買い物ができる

### Wan Chai / CWB
+ [x] ブルーハウス
  - 昔の香港を彷彿とさせる色とりどりの家が残る一角
  - ただそれだけである
+ [x] 太原街
  - アンティーク品を売ってるマーケット
  - 昼からやってる
+ [x] noonday gun
  - 正午きっかりに放たれる大砲
  - 毎日やってる
+ [x] Golden Bauhinia Square
  - コンベンションセンターの横
  - 花火、シンフォニー・オブ・ライツの穴場

### Kennedy town
+ [x] Kennedy town
  - 日曜の昼間に行ってゆっくり食事とビールを楽しむのが吉

### Stanley
+ [x] 赤柱（スタンレー）
  - 海沿いの小さな街
  - マーケットと、食事する場所くらいしかない

## Kowloon
### Tsim Sha Tsui
+ [x] プロムナード
  - 海沿いの道
  - デートスポットとしてはいいかも
+ [x] 九龍公園
  - かなり広い公園
  - ランニングをしているおじさんや、プールもある
  - 自然に囲まれてゆっくり散歩ができる

### Kowloon
+ [x] Sky100

## Others
+ [x] 香港ディズニーランド
  - 日本の半分程度の敷地なので回りやすい
  - ディズニーホテルのクリスタル・ロータスでは予約必須のキャラクター飲茶が人気
+ [x] Lamma island
  - シーフードの有名な香港で3番目に大きい島
  - ハイキングも可能（水分はハイク前に購入必須）
+ [x] Ngong Ping 360
  - 30分にも及ぶロープウェイが見所
  - 大仏にも会えます
+ [x] Cheung Chau island
  - 海鮮料理が安く食べられる
  - ハイキングコースがあり、距離も選べる（2~3hコースから、5hコースまで）
