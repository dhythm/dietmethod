# TOEIC380点の人間が英語で仕事をするために実施した学習法
## はじめに
僕は英語が苦手な人間です。  
学校教育の英語は詰め込み学習の要素が強く、単語をひたすらノートに書き込んで記憶することでなんとか大学まで進学しました。  
もちろん英語の楽しさを感じたことはありません。  
大学院では論文を英語で読むこともありましたが、やはり自分の中では英語は足枷でした。  

そんな状態で臨んだ初めてのTOEIC(会社の内定式で受けた)。  
ページをめくるタイミングも何もわからない僕は380点というクレイジーな点数を叩き出しました。  
その時、僕の隣でスラスラと問題を解いていた同期は今、海外で仕事をしています。  

そして自分はというと、今なんと海外で仕事をしています。  

ここでは、TOEIC380点だった人間が、英語ができるようになるために取り組んだ学習方法とその成果、  
そして海外赴任生活でどのように成長していくのかを記録していきます。  
(成長しない可能性もあり。ライザップじゃないけど、自分への監視も兼ねて。)  

## TL;DR
- 英語は異なる文化を知るためのツール(できなくても良いが、できると人生が豊かになる)
- 完璧になってから使おうとしない
- 学習を継続するためには目標じゃなく、目的を定める
- インプットとアウトプットの両方を意識する

## (英語)学習における問題
英語に限らずですが、新しいことをやろうとした時に必ずと言っていいほど問題となるのが、  

- 時間がない(忙しい)
- お金がかかる

ではないでしょうか。  
英語とは別にダイエットにもチャレンジしたことがある([別記事](https://github.com/dhythm/memorandom/blob/master/how_to_diet.md)参照)のですが、同じことを考えていました。  

### 時間について
人は必要に駆られれば本気で取り組むものです。  
時間がない、ということはその人にとっての英語学習がそれほどプライオリティが高くないということです。  

これは英語を習得する目的が曖昧であることが原因の一つではないでしょうか。  

大脳生理学/脳科学に「エフィカシー」という言葉があります。  
僕はこれを「自分は何ができるか・どうあるべきか」というセルフイメージみたいなものと捉えています。  

英語を使って何がしたいか、そして英語を習得したことで自分がどうなっているのかを明確に描けていないことが課題と考えます。  

「字幕なしで映画を観れるようになりたい」のような目標はよく見かけます。  
これは目標であって、目的ではありません。  
必要性に駆られてなければ、継続して学習を続けることはなかなか困難だと思います。  

プライオリティを上げる方法の一つとして、ライザップのようにお金をかける方法があります。  
安くはないお金を支払うことで強制的にプライオリティをあげてしまうことも、奥の手としてあると思います。  

### お金について
インターネットがある今の時代、(高い)お金を支払わなくてもできることは色々とあると思います。  
初期はお金をかけずに初めて、いいなと思ったら自己投資としてお金をかけた学習方法にシフトする、という作戦をとることもできるのではないでしょうか。  

自分の場合で一番お金をかけなかった方法が、「帰国子女の友達とのLINEを英語だけにする」でした。(詳細は後述)  

### 理想が高すぎる
上記の二つに加えて、語学で特に感じるのが、__理想が高すぎる__ ということです。  
「こんなレベルでは使うのが恥ずかしい」という意見を聞きます。  

幸い僕はそんな感覚は持ち合わせてませんでした。  
母国語である日本語ですら使い方を間違うことがあります。  
それなのに第二言語は「完璧じゃないと使っちゃダメ」なんておかしいと思いませんか？  

自分の英語がブロークン(※改善したいとは思っています)なのは理解した上で、まずは使ってみることが大事なんだと思います。  
ありがたいことに英語をできる人は優しく聞いてくれます。  
(外国の人がカタコトの日本語で話そうとしてくれるときと同じ感覚なのかもしれませんね)  

## 英語ができない人間が実践した英語学習法
前書きが長くなりましたが、僕が実践した英語学習方法を紹介します。  
ちなみに、2018/08/12時点の英語力はまだクソです。  

将来的にペラペラになったときに読み返して懐かしめたら嬉しいな、とか夢想しています。  
基本的に時系列順にやったことです。  

### オンライン英会話
金をかけないでも勉強は可能(ｷﾘｯ とか言ってたのにすいません。  

会社が高受講率をキープすることを条件に受講させてくれるプログラムを提供していました。  
[Bizmates](https://www.bizmates.jp/) ってサービスで、1日1回25分です。  
タダで受けられるならせっかくの機会だし、と思ってキャンペーンに乗っかりました。  

インターネットで手軽に、安価に英会話ができる時代ってのは本当にすごいと思います。  

ちなみにこの時の英語学習の目的は「費用を会社持ちにすること」でした。  
余談ですが、予約システムのバグを踏んでしまったらしく受講率が100％(227/226)を超えていたこともあります。  
2018/08/12時点で(399/400)で99.7％の受講率です。  

これにより「英語を話すこと」(※上手に話すことではありません)に慣れることができました。  
海外の人とコミュニケーションを取るためのツールとして意識できたので非常に良い学習方法だと思います。  

### LINE
帰国子女の友人に相談して、この時からずっと(今もなお)英語だけでチャットのやり取りをしています。  

高校からカナダに留学していた友人は、STARWARS の BB-8 の機械音の抑揚でだいたい何を言おうとしているのかわかると言っていました。  
僕にはちょっと何言っているのかわかりませんでした。  
ともあれ、アホな事につきあってくれる友人を持てたのはありがたい事です。  

友達との会話の中では「英語での表現」が学べます。  
ネイティブではない私は日本語で表現したい内容が浮かびます。  
それを英語で伝えようと思うとどうすれば良いか、と考えるようになりました。  

LINEのスピード感もオススメです。  

### 英語で考える(英語の人格をもつ)
社会人４年目の時に会社の制度でインストラクターとして新人教育を任せてもらったのですが、教え子が幼少期を海外(フランス？)で過ごしたガチモンの帰国子女でした。  

その子が普段日本語で会話している時のカタカナ文字の発音は日本人のそれと変わりありません。  
ある時、英語だけの文章を読んでみてくれた際に放たれた言葉・発音は、まさにネイティブのそれでした。  

真に英語ができる人は、日本語と英語で別の脳を持っている、と感じた一件でした。  
それ以来、自分の中に英語の人格・脳を持つように心がけています。  

### 英単語帳
DUO 3.0を買いました。  
載ってる例文のインパクトが強いので面白く読むことができます。  
日常でほぼ使わないクレイジーな表現しか覚えられません。  
```
"Natto" smells awful but tastes terrific.
```
なんて文章、一生使うことないんだろうなと思ってました。  
赴任先で納豆の話になってまんまコピーで使いました。人生何があるかわかりませんね。  

あと、音声CDも買って聞くようにしています。  
なんども聞き返してわかっているつもりでも、聞き取れない文章がたくさんあります。  
全ての基礎となるボキャブラリーの強化はまだまだこれからの課題です。  

赴任して強く感じるのは、ボキャブラリーの少なさです。  
相手の言葉を捉えても単語の意味が分からなければ置いてけぼりをくらいます。  

### Podcast
ランニング中に音楽を聴いていたのを、Podcast で英語のニュースを聞くようにしました。  
ニュースであれば何が起こっているのかもある程度推測できるので初心者でも取っつきやすいと感じます。  
日本の記事を英語で説明しているものであればなおさらです。  

副次的な効果ですが、ニュースを聞く時間と捉える事でランニングをサボる口実も奪われていきました。  
このおかげで「ダイエット」という目的と「英語学習」を絡めることに成功しました。  

### 英語の記事を読む
[Hacker News](https://news.ycombinator.com/) で気になった記事をいくつか読むことを日課にしました。  
Google Chrome の plugin に google翻訳や google dictionary があるので気になった単語を即座に調べられます。  
Tech系の情報収拾と、英語学習を並行して実施できるので個人的にはすごく気に入った勉強方法です。  

「Tech系の最新動向をキャッチアップする」という目的と、「市場価値のあるエンジニアとして成長する」という目標とを「英語学習」に絡めはじめました。  

### Twitter
Twitter で時たま思いついたことを英語でつぶやくことを始めました。  
下手くそな英語ですが、Twitter という媒体は気にせずつぶやくことができるのですごくいいツールだと思います。  

インプットだけじゃなく、アウトプットを意識することで自分の表現に幅が出たり、英語を使う事に躊躇しなくなっていくのかな、と感じています。  

### Youtube
Youtube の Tech系の動画を、自動生成の英語字幕付きで見ています。  
まだまだ聞き取れない単語は多いですが、字幕のおかげである程度は脳に入ってきます。  
並行して Tech系の技術勉強にもなるので、個人的にはポイントが高い学習方法です。  


## 赴任後の状況
上述の通り、もともと僕は英語力が皆無の人間です。  
そんな僕ではありますが、運よく海外赴任のお話をいただきました。(今年の２月)  

僕は呑気に、  
「海外拠点で、日本人も数人いる場所で、英語で仕事をするんだろう。  
　英語力も鍛えられるしありがたい話だな。」  
と考えていました。  

蓋を開けてみると、欧米人と香港人だけで構成された日本人ゼロの環境で、しかも Tech 的にもこれまでのバックグラウンドと真逆のお仕事でした。  
言葉だけでなく、スキルも未知の世界。正直焦ります。  

自分の中で、英語力強化はトッププライオリティに駆け上がりました。  
ここから先は自分でも未知の世界です。  
果たして僕は英語ができるようになるのでしょうか。  

### 赴任 0 ~ 1 ヶ月
同僚の英語が聞き取れなくて聞き返してばっかりです。めちゃくちゃ申し訳ない気持ちでいっぱいになる。  

知り合いのアドバイスでホワイトボードノートを持ってきているので、拙い英語と絵を描いて説明・議論を繰り返す日々。  
Daily Scrum で自分の近況を説明しないといけないため、英語で考えて発信する機会が毎日あります。  
これは非常にありがたい。使わないと伸びないです。  

一方で、メールや Slack での会話はリスニングと比べるとだいぶ助かります。  
当面の課題はリスニング力の強化（＋ボキャブラリー強化）です。  

language issue がある状態で大事なのは「スーパーマン」を目指すことではありません。  
「いてくれると助かる存在」が現実解だと思います。  
僕の場合は日本向けのプロダクト開発のために海外に赴任しています。  
日本の金融システムは海外の人から見ると「マジで？」と思うことが多々あります。  
自分にはそのバックグラウンドがあり、その部分を引き受けるアブソーバーのポジションで価値を出していこうと考えています。  


教科書みたいに全部が綺麗な英語じゃありません。  
誤字脱字もあるし、省略した言い回しとか、よくわからない表現・引用もあります。  
この辺は感覚で捉えるしかないんだな、と割り切っています。  

リスニング時には疑問詞を意識するようになりました。  
`What`, `When`, `How`...、何を質問されているのか、話の Subject が何かを意識します。  

また口頭で質問することが苦手ということに気づきました。  
文字で書くとちゃんと疑問文にできても、頭の中にある内容をちゃんと疑問系で質問するのが難しいです。  

#### 同僚がよく使う言葉
単語・イディオム|意味・ニュアンス
:---            |:---
`make sense`    |なるほどね！や、理解しました、など。かなり頻出。
`you guys`      |`you`を明確に複数の意図で利用する際。
`Sorry?`        |聞き返すとき。`pardon?`はほとんど使わない。


### 赴任 1 ~ 2 ヶ月
香港での銀行口座開設と、香港IDの取得をおこないました。  
当然ながら全て英語です。一応、リジェクトされない程度の英語力は身についているようです。  

(クセを掴んだ？のか、)同僚の英語は少し聞き取れるようになります。  
とはいえ、集中していないと全然ダメ。ヒートアップした会話もまだ聞き取れません。  
なので会議はかなり脳が疲れます。  
Skype会議で音声が途切れたり、複数人が同時に喋ったりすると Abort します。  

本業のプロダクト開発に着手したのですが、ここはスタートアップ企業。手とり足とりは教えてはくれません。  
(重要なことはちゃんと教えてくれます。日本の新人教育みたいに過保護ではない、というだけ。)  
当たり前ですが連携される資料は全て英語です。  
こればっかりはなんとかついていくしかありません。  
(わかるまで読み返す、夜に復習する、等)  

以前に、会議通訳者である高松珠子さんのインタビュー記事で、  

> 英語ができるようになるというのは、“大きな図書館の鍵”を渡されるようなものです。
> 一人ひとりの人生にいろんな可能性を与えてくれます。

という言葉を見ました。  
変化の多いIT系のエンジニアにとって、英語ができないことは大きなディスアドバンテージとなると感じます。  
一方で、英語ができると世界中の膨大な知識にもアクセスできるので、英語を学ぶ価値は非常に高いと思います。  

#### 同僚がよく使う言葉
単語・イディオム|意味・ニュアンス
:---            |:---
`Anyway`        |話の転換、話を戻す際に使います。`by the way`より汎用的。
`stuff`         |モノを指すときによく使われます。`staff`だと思っていたけど、意味を考えるとこっちです。
`definitely`    |デフネリー、と発音します。確信を持って話す際にかなりよく使われるイメージです。
